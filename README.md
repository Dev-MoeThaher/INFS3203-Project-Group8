# AI Travel Planner

AI Travel Planner is a web application that helps travelers quickly plan trips by generating AI-powered itineraries, packing lists, attraction suggestions, and local travel tips based on a destination, travel dates, and user preferences. It is designed for students, families, and casual travelers who want a fast and simple way to organize a trip.

## Features

- Create and save trip plans
- View previously saved trips
- Edit and delete trip records
- Responsive travel planning interface
- **AI-powered itinerary generation via Gemini**
- **AI-powered packing list generation via Gemini**
- **AI-powered local tips and attraction recommendations via Gemini**

## Tech Stack

- **Backend:** Node.js + Express  
  Chosen because it is lightweight, fast to set up, and suitable for building REST APIs.
- **Database:** MongoDB Atlas  
  Chosen because it is cloud-based, easy to integrate, and recommended in the project guidelines.
- **Frontend:** HTML + CSS + JavaScript  
  Chosen to keep the project simple and achievable within 3 weeks.
- **AI / LLM:** Google Gemini  
  Chosen because it supports itinerary and recommendation generation and matches the course requirement for AI-powered features.
- **Containerization:** Docker  
  Used to ensure the app runs consistently in all environments.
- **Version Control / Workflow:** GitHub, GitHub Issues, Pull Requests, Project Board
